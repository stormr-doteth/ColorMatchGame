# FirstCodexRoblox
Generated by [Rojo](https://github.com/rojo-rbx/rojo) 7.7.0-rc.1.

## Getting Started
Install dependencies and apply required patches:

```powershell
# PowerShell (recommended on Windows)
.\setup.ps1

# Git Bash / macOS / Linux
bash setup.sh
```

This runs `wally install` and patches TopBarPlus v3.4.0 to fix a deferred `Clone()` race condition. Always use the setup script instead of running `wally install` directly.

Build the place from scratch:

```bash
rojo build -o "FirstCodexRoblox.rbxlx"
```

Next, open `FirstCodexRoblox.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

The client bootstrap mounts a React Luau UI into `PlayerGui`.
Server data sessions are managed through `ProfileStore` via `src/server/data/ProfileService.luau`.

For more help, check out [the Rojo documentation](https://rojo.space/docs).
